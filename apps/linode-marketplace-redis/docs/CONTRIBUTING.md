# Contributing

Thanks for your interest in contributing to the Redis One-Click Cluster!

You can contribute by [opening an issue](https://github.com/akamai-compute-marketplace/redis-occ/issues/new) or submitting a pull request.

## Opening an issue

Feel free to open an issue to report a bug or request a feature.

## Submitting a pull request

1. Fork this repository.
2. Clone your fork to your local machine.
3. Create a branch from `develop`, e.g. `$ git checkout develop && git pull && git checkout -b feature/my-feature`.
4. Make your changes, commit them, then push them to your fork.
5. Open a pull request against `develop`.
